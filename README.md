## 🌍 World Models research

> I'm building agents that **learn to imagine** — training neural world models that simulate future states in a compact latent space, enabling planning without real-world interaction.

<div align="center">

[![latent-dynamics](https://github-readme-stats.vercel.app/api/pin/?username=gitmehdii&repo=latent-dynamics&theme=tokyonight&hide_border=true)](https://github.com/gitmehdii/latent-dynamics)
[![world-model-nav](https://github-readme-stats.vercel.app/api/pin/?username=gitmehdii&repo=world-model-nav&theme=tokyonight&hide_border=true)](https://github.com/gitmehdii/world-model-nav)

</div>

| Project | What it does | Key concept |
|---|---|---|
| 🌙 **dreamer-lite** | Reimplementation of DreamerV2's RSSM — agent learns in pure imagination | Recurrent State Space Model, KL with free bits |
| 🔮 **latent-dynamics** | VAE + LSTM/Transformer predicting future in latent space | β-VAE, NLL dynamics loss, autoregressive rollout |
| 🗺️ **world-model-nav** | Agent plans by simulating H-step trajectories before acting | Residual transition model, MPC in latent space |
